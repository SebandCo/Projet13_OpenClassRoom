Introduction
============

Le Projet 13 d'OpenClassRoom consiste à moderniser, conteneuriser, tester et déployer une application Django existante
L'objectif est de mettre en place une architecture incluant :
- Un pipeline CI/CD complète
- Une image Docker
- Un déploiement automatisé sur Render
- Une documentation technique sur Read the Docs

L'application permet de consulter des profils et des locations via une interface Django
